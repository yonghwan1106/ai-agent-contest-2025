# 내일(My Job) 찾기 내비게이터 - 필요한 서비스 목록

## 공공 서비스

### 1. 고용보험 관련 (고용노동부 / 고용24)
- **서비스명**: 고용보험 피보험자격 조회 및 이직확인서 발급
- **기관**: 고용노동부
- **플랫폼**: 고용24 (www.work24.go.kr)
- **기능**: 4대보험 가입이력 조회, 퇴사 사실 확인, 이직확인서 발급
- **API 연동**: 가능 (행정정보 공동이용)

### 2. 실업급여 신청 (고용노동부 / 고용보험)
- **서비스명**: 실업급여 수급자격 인정 및 신청
- **기관**: 고용노동부 고용센터
- **플랫폼**: 고용보험 홈페이지 (www.ei.go.kr)
- **기능**:
  - 수급자격 자동 확인 (피보험 기간, 비자발적 이직 여부)
  - 구직급여 신청
  - 실업인정 신청 및 구직활동 내역 제출
- **API 연동**: 가능

### 3. 워크넷 구직등록 (고용노동부 / 워크넷)
- **서비스명**: 구직등록 및 채용정보 매칭
- **기관**: 고용노동부 한국고용정보원
- **플랫폼**: 워크넷 (www.work.go.kr)
- **기능**:
  - 구직자 등록 및 이력서 등록
  - AI 기반 채용정보 추천
  - 온라인 입사지원
  - 구직활동 내역 관리
- **API 연동**: 가능 (워크넷 OpenAPI)

### 4. 국민내일배움카드 (고용노동부 / 직업훈련포털)
- **서비스명**: 국민내일배움카드 발급 및 훈련과정 신청
- **기관**: 고용노동부
- **플랫폼**: 직업훈련포털 HRD-Net (www.hrd.go.kr)
- **기능**:
  - 내일배움카드 발급 신청
  - 직업훈련과정 검색 및 추천
  - 훈련과정 수강신청
  - 훈련비 지원 신청
- **API 연동**: 가능 (HRD-Net OpenAPI)

### 5. 취업성공패키지 (고용노동부 / 고용센터)
- **서비스명**: 취업성공패키지 참여 및 관리
- **기관**: 고용노동부 고용센터
- **플랫폼**: 고용24
- **기능**:
  - 취업성공패키지 참여 신청
  - 개인별 취업활동계획 수립
  - 집중 취업지원 프로그램 안내
- **API 연동**: 가능

### 6. 국민취업지원제도 (고용노동부)
- **서비스명**: 국민취업지원제도 신청 및 관리
- **기관**: 고용노동부
- **플랫폼**: 고용24
- **기능**:
  - 지원대상 자격 확인
  - 취업지원서비스 신청
  - 구직촉진수당 신청
- **API 연동**: 가능

### 7. 국가자격증 조회 (한국산업인력공단 / 큐넷)
- **서비스명**: 국가자격증 정보 조회 및 응시 신청
- **기관**: 한국산업인력공단
- **플랫폼**: 큐넷 (www.q-net.or.kr)
- **기능**:
  - 보유 자격증 조회
  - 추천 자격증 정보 제공
  - 시험 일정 안내 및 응시 신청
- **API 연동**: 가능 (큐넷 OpenAPI)

### 8. 4대보험 정보연계센터
- **서비스명**: 4대보험 가입내역 통합 조회
- **기관**: 국민건강보험공단, 국민연금공단
- **플랫폼**: 4대사회보험 정보연계센터
- **기능**:
  - 실시간 4대보험 가입 상태 조회
  - 퇴사 감지 (자동 트리거)
- **API 연동**: 가능 (행정정보 공동이용)

## 민간 서비스

### 1. 채용정보 플랫폼
#### 사람인 (www.saramin.co.kr)
- **기능**: 채용공고 검색, 온라인 지원, 입사지원 현황 관리
- **API 연동**: 가능 (사람인 채용정보 API)

#### 잡코리아 (www.jobkorea.co.kr)
- **기능**: 채용공고 검색, AI 매칭, 이력서 관리
- **API 연동**: 가능

#### 인크루트 (www.incruit.com)
- **기능**: 채용정보 제공, 기업정보 조회
- **API 연동**: 가능

### 2. 캘린더 앱 (일정 관리)
#### 구글 캘린더
- **기능**:
  - 실업인정일 자동 등록
  - 면접 일정 관리
  - 훈련과정 일정 동기화
  - 구직활동 마감일 알림
- **API 연동**: Google Calendar API

#### 네이버 캘린더
- **기능**: 동일
- **API 연동**: 네이버 캘린더 API

### 3. 문서 관리 앱
#### 구글 드라이브
- **기능**:
  - 이력서, 자기소개서 자동 저장
  - 면접 후기 메모 저장
  - 증빙서류 백업
- **API 연동**: Google Drive API

### 4. 알림 서비스
#### 카카오톡 알림톡
- **기능**:
  - 실업인정일 D-7, D-3, D-day 알림
  - 구직활동 제출 독촉 알림
  - 추천 채용공고 알림
  - 훈련과정 수강 알림
- **API 연동**: 카카오 비즈니스 API

### 5. 위치 기반 서비스
#### 카카오맵 / 네이버 지도
- **기능**:
  - 가까운 고용센터 위치 안내
  - 면접 장소 길찾기
  - 훈련기관 위치 안내
- **API 연동**: 카카오맵 API, 네이버 지도 API

### 6. AI 챗봇 플랫폼 (서비스 구현 기반)
#### ChatGPT / Claude / Gemini
- **기능**:
  - 자연어 대화 처리
  - 이력서 분석 및 직무역량 파악
  - 맞춤형 훈련과정 추천
  - 면접 준비 조언
- **API 연동**: OpenAI API, Claude API, Gemini API

### 7. 간편인증
#### PASS (통신사 인증)
- **기능**: 본인 인증, 전자서명
- **API 연동**: PASS 인증 API

### 8. 결제 서비스
#### 네이버페이 / 카카오페이
- **기능**: 훈련비 결제 (자부담분)
- **API 연동**: 네이버페이 API, 카카오페이 API

## 데이터 연계 필요 사항

### 자동 감지를 위한 데이터
1. **4대보험 DB** (국민건강보험공단)
   - 피보험자격 상실 정보 (퇴사 감지)
   - 실시간 또는 일 1회 연동

2. **고용보험 DB** (고용노동부)
   - 피보험 기간 조회
   - 수급자격 판정 기준 데이터

### 통합 인증
- **정부24 통합인증**
- **카카오/네이버 소셜 로그인**
- **PASS 인증**

## 서비스 흐름별 필요 서비스 매핑

| 단계 | 공공서비스 | 민간서비스 |
|------|------------|------------|
| 1. 자동감지 | 4대보험 정보연계센터 | 카카오톡 알림톡 |
| 2. 실업급여 신청 | 고용보험(실업급여), 고용24 | PASS 인증, 구글 드라이브 |
| 3. 구직등록 | 워크넷 | 사람인/잡코리아, 구글 드라이브 |
| 4. 훈련과정 추천 | HRD-Net(내일배움카드), 큐넷 | ChatGPT/Claude(AI 분석) |
| 5. 구직활동 관리 | 워크넷, 고용24 | 사람인/잡코리아, 구글 캘린더, 카카오맵 |
| 6. 실업인정 | 고용보험(실업인정) | 카카오톡 알림톡, 구글 캘린더 |
