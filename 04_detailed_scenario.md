# 세부 시나리오 (AI 대화창)

## 사용자 프로필 (예시)
- **이름**: 김민준 (32세, 남성)
- **상황**: IT 중소기업에서 3년 근무 후 회사 구조조정으로 퇴사 (비자발적 이직)
- **경력**: 웹 개발자 (JavaScript, React 사용)
- **희망**: 더 나은 조건의 회사로 이직, 필요시 최신 기술(AI, 클라우드) 추가 학습 의향 있음

```
[1단계] 퇴사 자동 감지 → AI가 먼저 연락

   ↓

   ↓

   ↓
[6단계] 실업인정 자동화
   ↓
[7단계] 재취업 성공 → 후속 지원
```

## 시나리오 핵심 특징

### 1. **자동 감지 (Proactive)**
- 사용자가 검색하기 전에 **AI가 먼저 퇴사를 감지**하고 연락

### 2. **원스톱 처리**
- **6개 플랫폼**을 오갈 필요 없이 **하나의 AI 에이전트**로 통합

### 3. **자동 양식 작성**
- 복잡한 신청서를 **AI가 자동으로 작성** (사용자는 확인만)

### 4. **맞춤형 추천**
- 이력서 분석 → **개인 맞춤형 훈련과정, 채용정보** 추천

### 5. **구직활동 자동 증빙**
- 채용 지원, 훈련 수강 등을 **자동으로 기록** → 실업인정 시 자동 제출

### 6. **지속적 지원**
- 재취업 후 **조기재취업수당**까지 챙겨주는 **끝까지 함께하는 AI**

## 기술 구현 포인트

### 사용된 기술/서비스
1. **4대보험 DB 연동** (퇴사 감지)
2. **고용24, 고용보험 API** (실업급여, 이직확인서)
3. **워크넷 OpenAPI** (구직등록, 채용정보)
4. **HRD-Net API** (내일배움카드, 훈련과정)
5. **사람인/잡코리아 API** (민간 채용정보)
6. **AI 분석** (ChatGPT/Claude) - 이력서 분석, 자기소개서 작성, 적합도 매칭
7. **카카오톡 알림톡** (알림)
8. **구글 캘린더 API** (일정 관리)
9. **PASS 인증** (본인 인증)

### 차별화 포인트
- **분산된 서비스 → 통합 여정(Journey)**
- **수동 신청 → 자동 감지**
- **일반 안내 → AI 맞춤 추천**
- **서류 작성 → 자동 생성**
- **사후 관리 부재 → 재취업 후까지 지원**
