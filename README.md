## 📋 프로젝트 개요

**실업부터 재취업까지, AI가 함께하는 새로운 시작**

퇴사 직후부터 재취업까지의 전 과정을 AI 에이전트가 자동으로 감지하고 순차적으로 안내하는 통합 고용 지원 서비스입니다.

### 핵심 가치

- **자동 감지**: AI가 퇴사를 먼저 감지하고 연락
- **원스톱 처리**: 6개 플랫폼 → 하나의 AI 에이전트로 통합
- **맞춤형 지원**: AI 기반 직무역량 분석 및 개인화 추천
- **끝까지 함께**: 재취업 후 조기재취업수당까지 안내

## 🎯 시나리오 특징

### 1. Proactive (선제적)
사용자가 검색하기 전에 **AI가 먼저 퇴사를 감지**하고 연락합니다.

### 2. Seamless (끊김 없는)
6개 이상의 플랫폼을 오갈 필요 없이 **하나의 AI 에이전트**로 모든 절차를 처리합니다.

### 3. Personalized (맞춤형)
- 이력서 AI 분석 → 직무역량 파악
- 개인 맞춤형 직업훈련 추천
- 적합도 기반 채용정보 매칭

### 4. Automated (자동화)
- 복잡한 신청서 자동 작성
- 구직활동 자동 증빙
- 실업인정 자동 제출

### 5. Continuous (지속적)
재취업 후 **조기재취업수당**까지 챙겨주는 끝까지 함께하는 AI
